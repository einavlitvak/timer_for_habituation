# Habituation Timer

A mobile-friendly, high-precision Progressive Web App (PWA) designed for habituation tasks, supporting delay timers, repeating interval alerts, sound synthesis, vibration support, and hands-free voice notes.

👉 **Live App URL**: [https://einavlitvak.github.io/timer_for_habituation/](https://einavlitvak.github.io/timer_for_habituation/)

## Features

* **Custom Timers**: Set total duration, initial start delay, and repeating interval alerts.
* **Responsive Visuals**: Glowing concentric progress rings tracking overall session and interval time.
* **Offline Operation**: Works fully offline once added to your phone's home screen.
* **Multi-Modal Alerts**: Custom Web Audio synthesized chimes/beeps and physical device vibrations (silent-friendly).
* **Wake Lock Toggle**: Toggles to prevent your mobile screen from turning off during a session.
* **Voice-Transcribed Notes**: Automatic speech-to-text dictation at intervals to log observations.
* **Live Action Log**: Displays session triggers and voice transcripts in a visual log.

## Installation

### Mobile (PWA)
1. Open the **[Live Link](https://einavlitvak.github.io/timer_for_habituation/)** on your phone's web browser.
2. Add it to your home screen:
   * **iOS (Safari)**: Tap the **Share** button and select **Add to Home Screen**.
   * **Android (Chrome)**: Tap the **Install** banner on-screen or select **Add to Home screen** from the menu.
